Software Components
===================

There are a number of open-source software modules being developed as part of
the ToMCAT project:

* `Minecraft-based virtual testbed <https://github.com/ml4ai/tomcat>`_.
* `Automatic Speech Recognition (ASR) Agent <https://github.com/ml4ai/tomcat-ASR_Agent>`_
* `Vocalic feature extraction, sentiment analysis, personality trait
  detection <https://github.com/ml4ai/tomcat-speechAnalyzer>`_.
* `Natural language processing (text) <https://github.com/clulab/tomcat-text>`_.
* `Multimodal sentiment analysis and personality trait detection <https://github.com/clulab/tomcat-speech>`_.
* `Face analysis <https://github.com/ml4ai/tomcat-faceAnalyzer>`_.
* `ToMCAT ASI (Artificial Socially Intelligent) Agent <https://github.com/ml4ai/tomcat-tmm>`_.
* `Dashboard <https://github.com/ml4ai/tomcat-dashboard>`_.
* `Semantic coordination detection component <https://github.com/ml4ai/tomcat-CDC>`_
